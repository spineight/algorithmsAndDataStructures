Полезные ссылки для дальнейшего изучения ML
1. Базовое машинное обучение

Популярные онлайн-курсы от ВШЭ (платить не нужно, видео доступны по ссылкам /learn/...):

    Курс «Введение в машинное обучение»: https://www.coursera.org/learn/vvedenie-mashinnoe-obuchenie
    Специализация «Машинное обучение и анализ данных»: https://www.coursera.org/specializations/machine-learning-data-analysis, состоит из курсов:
        «Математика и Python для анализа данных»: https://www.coursera.org/learn/mathematics-and-python
        «Обучение на размеченных данных»: https://www.coursera.org/learn/supervised-learning
        «Поиск структуры в данных»: https://www.coursera.org/learn/unsupervised-learning
        «Построение выводов по данным»: https://www.coursera.org/learn/stats-for-data-analysis

Курс видеолекций К.В. Воронцова Школы Анализа Данных Яндекса (Осторожно, лекции расчитаны на студентов 3-4 курсов)

    https://yandexdataschool.ru/edu-process/courses

2. Нейронные сети

    Очень крутой курс про нейронные сети (на английском) http://cs231n.stanford.edu/syllabus.html
    Аналогичный курс с акцентом на анализ текстов и рекуррентные сети (на английском) http://cs224d.stanford.edu/syllabus.html
    Про то, как работают рекуррентные сети http://karpathy.github.io/2015/05/21/rnn-effectiveness/
    Потрясающий блог с очень красивыми, простыми и познавательными визуализациями сложных моделей http://colah.github.io/

3. Прочие ресурсы

    http://www.machinelearning.ru/ – ресурс с полезными ссылками и теорией машинного обучения
    https://www.kaggle.com/ – соревнования по машинному обучению
    https://archive.ics.uci.edu/ml/datasets.html – архив с датасетами для машинного обучения
