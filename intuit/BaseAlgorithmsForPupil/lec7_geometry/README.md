### Геометрия
Геометрия традиционно считается достаточно сложной темой  

Мы с вами начнем знакомиться с геометрическими примитивами, которые в дальнейшем позволят создать более сложные
геометрические алгоритмы.  

### Базовые понятия
Давайте начнем с самых базовых геометрических примитивов  
Мы будем заниматься двухмерной геометрией, тоесть на плоскости.  
#### Точка
То без чего нельзя вообще жить это точка.  
Точка неопределяемое понятие, тоесть нельзя сказать, что точка это то-то  
Но если введена система координат, то точка тогда задается своими координатами.

#### Вектор
Вектор - такой геометрический объект, он характеризуется двумя параметрами: направление и длина.  
Можно говорить, что вектор - это отрезок которому задали какое-то направление  

Давайте мы введем у нас на нашей плоскости на которой мы работаем систему координат  
прямоугольная Декартовая система координат, в ней точка характеризуется двумя числами  `x,y`
и ее можно в этом смысле записать как пару `(x,y)`  

Вектор в системе координат тоже характеризуется двумя числами - отложим этот отрезок от точки `(0,0)`, тогда  
его конец окажется в какой-то точке, вот эта точка и называется координатами вектора
```c++
                           ^Y
                           |   (x1,y1) -- координаты вектора
                           |   ^
                           |  /
                           | / 
          _________________|/________________________>X
                           |
                           |
                           |
                           |
                           |
```  

Не спроста мы изучаем точку и вектор в паре, оба характеризуются двумя числами, при этом особых различий с 
принципиальной точки зрения между точкой и вектором (в смысле представления в программе) нету

## Прямая
Можно считать неопределяемым, можно определяемым понятием, если исповедовать алгебраический подход  
Через любые две точки можно провести прямую  
Если у нас введена координатная сетка, то можно изучать координаты точек лежаших на прямой.  
Рассмотрим точку на прямой с координатами x,y  
Все точки лежащие на прямой удволетворяют какому-то условию, это условие называется уравнением прямой  
Есть несколько способов записать такие условия:  
1. Мы выясняли когда говорили о скалярном и векторном произведение, что если взять вектор, то
множество векторов которые сонаправлены и лежат на одной прямой с этим вектором  - 
векторное произведение будет равно 0  

Допуситм у нас есть точки p(x1,y1) и q(x2,y2) которые задают прямую,  
Давайте выясним множество векторов, чтобы векторное произведение было равно 0  
Какие координаты будут у вектора проведенного из точки p в точку q:  
r(x2-x1, y2-y1)  

```c++

                   \
   r(x2-x1,y2-y1)   ^. (x2,y2)
                     \
                      \
                       \. (x1,y1)
                        \
                         \
                          V . (x,y)
                           \

 (x-x1, y - y1) X (x2-x1, y2-y1) = 0   (векторное произведение, если на прямой лежат вектора, то синус равен 0 поэтому векторное)


(x - x1)(y2 - y1) - (y - y1)(x2 - x1) = 0
(y2 - y1)x + (x1-x2)y = (y2-y1)x1 + (x1-x2)y1
```
Итого мы получили какое-то условие, которому должны удволетворять x и y,   
чтобы лежать на прямой  `(x2,y2) (x1,y1)  `
```c++
Давайте обозначим y2-y1 = a; x1-x2 = b

ax + by = ax1 + by1 - мы получили какое-то выражение ax + by, которое для всех точек прямой одно и тоже

ax1 + by1 обозначим как c, тогда:
ax + by = c ( однородное уравнение прямой )
```
Верно ли, что у каждой прямой одно однородное уравнение?
```c++
задают одну и ту же прямую
2x + 3y = 1
6x + 9y = 3  
```
Чтобы сделать однозначно(почти), можно потребовать
```
a^2+b^2=1 - это значит что вектор который мы выбрали на прямой(направляющий вектор прямой) нормирован

y2-y1 = a; x1-x2 = b
```
Тогда получится уравнение с точностью до знака

В этом смысле, удобно рассматривать, так называемую ориентированную прямую  
Посмотрим, какие у нас координаты направляющего вектора:
```c++
(y2-y1, x2-x1)
так как мы приняли y2-y1 = a; x1-x2 = b
то направляющий вектор у нас имеет координаты:
(-b,a)
```

Можно ввести такое понятие как направленная прямая
Направленая прямая - хороша тем, что если встать по направлению прямой,  
то можно понять какие точки слева какие справа  

Проверить, что точка лежит на прямой - подставить в уравнение и посмотреть, что равно 0   
Если прямая задана направляющим вектором, то можно сразу взять векторное произведение с направляющим вектором и посмотреть, что он равно 0  
Чтобы задать прямую однозначно нужны направляющий вектор и точка на этой прямой  


###Способы хранить прямую
https://ru.wikipedia.org/wiki/Прямая#Общее_уравнение_прямой
1) в однородном уравнении (хранить a,b,c)  
с их помощью мы можем вычислить все остальное:  
направляющий вектор (-b,a)
Точку на прямой, она имеет координаты: `(ac/(a^2+b^2), bc/(a^2+b2))`,    
причем это не просто абы какая-то точка,  
это самая лучшая точка на этой прямой (эта точка - основание перпендикуляра опущенного из начала координат на нашу прямую,
это ближайшая к началу координат точка на нашей прямой)    

Подставим ее координаты в уравнение прямой:
```c++
a * ac/(a^2+b^2) + b * bc/(a^2+b2) = c
a^2*c/(a^2+b^2) + b^2*c/(a^2+b^2) = c
(a^2+b^2)*c/(a^2+b^2) = c
c = c
```
Чтобы найти c - нужно знать какую-нибудь точку на прямой
```c++
c = a*x1 + b*y1
```


Все параллейные прямые имеют одинаковые направляющие вектора, чтобы однозначно знать прямую - еще нужна точка на этой прямой  

2) Точку на прямой x1,y1 и направляющий вектор(dX, dY)
```
                    x2 = x1 + dX; y2 = y1 + dY
                    a = dY; b = -dX; c=   ax1 + by1
```

Как посчитать расстояние от точки до прямой:  
```
        D \      /
           \    /
     A(x,y)  \ /.C (x2,y2)
     x \      /
          \  /
            /
           /.B (x1,y1)
          /
```

пусть прямая заданна направляющим вектором(dX,dY) и точкой(x1,y1)
Тогда длина этого перпиндикуляра: Длине вектора BA * sin(ABC)
Если направляющий вектор прямой нормирован, тоесть dX^2 + dY^2 = 1  
То векторное произведение = BA X BC = |BA| * |BC| * sin(ABC) = |BA| sin(ABC) (так как длина BC=1, он нормирован)
Тогда заметим,что |BA| * sin(ABC) - и есть интересующее нас расстояние  
Распишем его через формулу векторного произведения, через координаты векторов BA(x-x1,y-y1) и BC(-b,a)
BA x BC = (x-x1) * a + (y-y1) * b

Чтобы найти расстояние от точки до прямой, нужно взять векторное произведение напрявляющего вектора(x-x1, y-y1)

```c++
    (x-x1)*dy - (y-y1)dx
```
```c++
(x-x1,y-y1) X (dx,dy) = (x-x1)dy - (y-y1)dx = xdy - x1dy - ydx + y1dx  
перейдем к однородным координатам: dy = y2-y1 = a; dx = -(x1-x2) = -b
```

### Параметрическое уравнение прямой
```
x = x1 + dX*t
y = y1 + dY*t  где t in [-inf, inf]

```

### Луч
Это когда мы на прямой взяли точку и только часть прямой по одну сторону от этой точки  
Луч удобно задавать началом и своим направляющим вектором  
Как проверить, что точка лежит на луче  
1. Лежит на прямой (используем векторное произведение)    
2. Вектор образованный началом луча и этой точки сонаправлен с вектором луча (используем скалярное произведение)


### Отрезок
Возмем две точки и прямую проходящую через них, возьмем только ту часть которая лежит между точками  
```
 x1,y1        x2,y2
  -------------
```
Отрезок можно хранить либо:  
- как две точки
- как одну точку и вектор который выходит из этой точки
- как одну точку + нормированный направляющий вектор + длину

Длину отрезка считаем по теореме Пифагора  sqrt((x2-x1)^2 + (y2-y1)^2)

Как проверить, что точка лежит на отрезке  
1. Проверяем что лежит на прямой
2. Два способа

первый - проверяем что лежит на луче AB, и на луче BA;

второй - когда точка ледит внутри ограничивающего прямоугольника (bounding box) этого отрезка  
Если у нас есть какой-то объект геометрический, то у него есть так называемый ограничивающий прямоугольник  

```
     |-------------|
     | \------|    |
     |  \     |____|
     |   \     ___/|
     |    \___/    |
      --------------
```

Точка лежит на отрезке тогда и только тогда: когда она лежит на прямой + лежит внутри ограничивающего прямоугольника этого отрезка
```c++

     -----|    y2
     |\   | 
     | \  | 
     |  \ |
     |___\|    y1
    x1     x2
это значит min(x1,x2) <= x <= max(x1,x2) && min(y1,y2) <= y <= max(y1,y2)
```     

### Задача пересечение прямых
Пусть есть две прямые давайте их как-нибудь зададим  
например в однородных координатах:  
```
a1x + b1y = c1
a2x + b2y = c2
```

Найти точку пересечения этих прямых  
У нас линейная система, два уравнения с двумя неизвестными   

Обозначим сначала как выглядит решение, а потом обсудим как его получить из геометрических соображений
```
d = a1b2 - a2b1   (векторное произведение векторов: (a1,b1) X (a2,b2))


Утверждается, что x = (c1b2 - c2b1) / d
                  y = (a1c2 - a2c1) / d
```

Давайте попробуем их вывести (хотя Станкевич может только порекоммендовать их запомнить, выводить их каждый раз долго)  
Как запомнить, нарисуем такую матрицу  
```c++
  ( a1  b1 )
  ( a2  b2 )
 У матрицы есть определитель(считается крест на крест):
  | a1  b1 |
      X
  | a2  b2 |

d = a1*b2 - a2*b1

Чтобы найти координаты x и y (используется формула Крамера)


     | c1 b1 |
 X = | c2 b2 |
     ----------
     | a1 b1 |
     | a2 b2 |

     | a1 c1 |
 Y = | a2 c2 |
     ----------
     | a1 b1 |
     | a2 b2 |
```

d = 0 - когда прямые паралельны - значит прямые либо не пересекаются либо совпадают
