### Окружность
Однозначно задается точкой центра и ее радиусом

#### Уравнение окружности
```(x0,y0) - центр
   r - радиус
(x-x0)^2 + (y-y0)^2 = r^2
```

```c++
Угол QRP - прямой
RP = sqrt(PQ^2 - RQ^2)
Опустим перпендикуляр RS на QP

Из их соотношения
RS / RQ = RP / QP
RS =  RP * RQ / QP

Задача составить уравнение прямой (касательной) проходящей через точки R,P

QS - это длина отрезка на который нам надо сместиться
x3 = x0+(x1-x0) * QS/QP
```

##### Определить выпуклый ли наш многоугольник
Если многоугольник простой и без самопересечений,то
нам достаточно обойти весь многоугольник и посмотреть знак векторного произведения
между соседними ребрами, если оно все время будет одного знака, то у нас многоугольник выпуклый

##### Определение нахождения точки внутри простого многоугольника
Внимание!!! Заранее нужно проверить не лежит ли точка на границе многоугольника  
Несколько способов  
* Способ которым никто не пользуется:  
смотрим под каким углом из точки видно каждое ребро  
(проводим отрезки к концам этого ребра и смотрим какой это угол)  
Если сумма углов получилась 2PI - точка лежит внутри (возможна проблема с точностью)  

* Метод дополнительного луча
Мы проводим луч из нашей точки строго вправо  
Считаем кол-во пересеченний с границами многоугольника   
Если этот луч пересекает границы многоугольника 0 или четное число раз - тогда точка вне
Если нечетное, тогда во многоугольнике  
В общем случае: идем по всем ребрам, смотрим пересекается ли луч с нашим ребром  

```c++
count = 0;
for(int i = 0; i < n-1; ++i) {
  if(not intersect P[i]P[i+1]) continue;
  if(точки ребра не лежат на нашем луче) {
    ++count
    continue;
  }
  Мы проверяем является ли та точка, с которой мы пересекаемся нижней в нашем ребре
```
